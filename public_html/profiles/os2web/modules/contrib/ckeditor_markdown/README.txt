CKEditor Markdown
=================

Description
===========
The Markdown plugin provides the ability to use markdown to style the content
in your CKEditor field.

Usage
=====
Go to the CKEditor settings (/admin/config/content/ckeditor) and
add the Markdown Button to any CKEditor profile you want. You also have to
enable the plugin under Editor Appearance >> plugins.
